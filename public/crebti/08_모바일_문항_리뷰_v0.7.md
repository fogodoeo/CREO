# CREBTI 모바일 문항 리뷰 v0.7

검토일: 2026-07-18  
상태: 내용 타당도·가독성 검토 완료 / 통계적 신뢰도 검증 전

## 검토 기준

- 한 문항은 E/I, S/N, T/F, J/P 중 한 축만 비교한다.
- 각 축은 서로 다른 상황의 3문항으로 반복한다.
- 두 답은 같은 질문에 답하며 길이와 구체성이 비슷해야 한다.
- 한쪽 답만 더 책임감 있거나 전문적으로 보이지 않게 한다.
- 모바일에서 질문은 한 번에 읽히고 답은 한 줄 또는 두 줄 안에 들어오게 한다.
- 문항 제목과 측정 축은 화면에 노출하지 않는다.

## 최종 구성

| 축 | 문항 | 확인하는 선호 |
|---|---|---|
| E/I | C01, C05, C09 | 대화로 정리 / 혼자 정리 |
| S/N | C02, C06, C10 | 현재 확인 정보 / 성장 가능성과 연결 |
| T/F | C04, C07, C11 | 조건과 근거 / 마음과 개인적 가치 |
| J/P | C03, C08, C12 | 미리 결정·구조화 / 다시 비교·상황 조정 |

## 이번 수정에서 해결한 부분

- C01: 경매 마감이라는 압박보다 생각을 정리하는 방식을 직접 비교한다.
- C05: 부정적으로 읽히던 `예상과 다른 변화`를 `반가운 변화`로 바꿔 양쪽 답의 정서 조건을 맞췄다.
- C06: `달라진 부분 / 달라질 모습`을 `실제 색·체형 / 앞으로의 가능성`으로 구체화했다.
- C08: 준비성의 우열로 보이지 않도록 `미리 완성 / 기본 세팅 후 조정`을 모두 정상적인 준비 방식으로 표현했다.
- C10: 모호했던 `한 마리씩 / 반복되는 특징`을 `확인된 특징 / 유전 흐름과 가능한 조합`으로 분명하게 나눴다.
- 전체 선택지의 마침표와 문장 형식을 통일하고, 화면의 중복 제목과 안내 문장을 제거했다.

## 해석 제한

이 질문지는 크레스티드게코 취향을 소재로 만든 재미용 분류다. 현재 검토는 문장과 축 구성에 대한 내용 타당도 검토이며, 실제 응답 데이터를 이용한 문항 판별력·내적 일관성·재검사 신뢰도는 아직 검증되지 않았다. 운영 데이터가 쌓이면 축별 응답 쏠림과 문항 간 상관을 확인해 문항을 교체한다.
